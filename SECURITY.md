# Security policy

Figment is a research/simulation project with no network, authentication, or
data-persistence surface: it runs local, deterministic games from seeded RNG.

If you nonetheless discover a security-relevant issue (for example in a
dependency or the CI configuration), please open a private report via GitHub
Security Advisories on this repository, or email the maintainer listed in the
profile. Please do not disclose it in a public issue until it has been addressed.
